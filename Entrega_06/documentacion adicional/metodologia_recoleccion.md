# Metodología de Recolección de Datos

## Objetivo

El objetivo de la recolección de datos fue construir una base de información confiable que permitiera analizar la evolución del circo desde una perspectiva histórica, normativa y cultural.

Debido a que no existe una única fuente que reúna toda esta información, fue necesario desarrollar una metodología propia de recopilación, organización y validación de datos.

---

# Proceso de recopilación

La información fue obtenida mediante investigación documental y consulta de fuentes públicas.

Se recopilaron antecedentes provenientes de:

- legislación nacional e internacional;
- organismos públicos;
- sitios oficiales de compañías circenses;
- artículos periodísticos;
- publicaciones académicas;
- instituciones culturales;
- archivos históricos.

Cada registro fue incorporado únicamente después de verificar su consistencia con al menos una fuente confiable.

---

# Construcción de bases de datos

Los antecedentes recopilados fueron organizados en archivos CSV independientes según la temática analizada.

Las principales bases desarrolladas fueron:

- Evolución normativa internacional.
- Industria circense.
- Casos emblemáticos.
- Sector circense chileno.
- Comparación entre modelos tradicionales y contemporáneos.
- Evolución generacional de compañías circenses.

Esta separación permitió trabajar cada dimensión de manera independiente y posteriormente integrarlas dentro de la narrativa de la webstory.

---

# Limpieza de datos

Antes de construir las visualizaciones se realizó un proceso de depuración de la información.

Las principales tareas fueron:

- eliminación de registros duplicados;
- corrección de nombres inconsistentes;
- normalización de categorías;
- revisión de fechas;
- homogeneización de variables;
- organización cronológica.

Estas tareas permitieron asegurar consistencia entre las distintas bases.

---

# Validación

La información fue contrastada mediante distintas fuentes públicas cuando existían discrepancias.

En aquellos casos donde no fue posible obtener una fecha exacta o información completa, el registro fue mantenido indicando explícitamente dicha limitación.

---

# Criterios de inclusión

Para formar parte de las bases de datos, los registros debían cumplir al menos uno de los siguientes criterios:

- relevancia histórica;
- importancia normativa;
- impacto cultural;
- representación del modelo tradicional;
- representación del modelo contemporáneo;
- disponibilidad de información verificable.

---

# Limitaciones

La investigación no constituye un catastro completo del circo mundial.

Las bases representan una muestra construida específicamente para responder la hipótesis planteada durante el proyecto.

Asimismo, algunos antecedentes históricos presentan diferencias entre fuentes, situación que fue documentada durante la recopilación.

---

# Conclusión

La metodología utilizada permitió construir bases de datos consistentes y transparentes, suficientes para respaldar las visualizaciones desarrolladas durante la investigación y responder las preguntas planteadas al inicio del semestre.
